# Sentiment Analysis

## Overview
This app predicts sentiments of a text (negative/positive).

## Tech Stack
- Python
- Scikit-learn
- NLP (TF-IDF, Logistic Regression)
- Streamlit

## Features
- Text classification
- Real-tie prediction
- Web app deployment

# How to Run
streamlit run app.py

# Deployment
Streamlit Cloud# sentiment-analysis
